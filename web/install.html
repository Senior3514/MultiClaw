One-command install

Install MultiClaw like a real product.

Copy one calm base command first. Layer provider, runtime, and extra surfaces after the base product is already in place.

Flow

A clean order of operations.

Install the core first. Configure AI next. Add networking, channels, and extra surfaces after the runtime is already alive.

1. Install core

Bring MultiClaw onto the machine with the smallest possible cognitive load.

2. Configure AI

Choose provider, model, API key, and bind mode after the base product is already installed.

3. Layer extras later

Add Tailscale, channels, and advanced integrations only after the runtime is working.
