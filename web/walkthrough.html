Walkthrough

One clean path from install to running product.

Install MultiClaw, configure the runtime, start it, verify it, and keep a clear next step at every stage.

1. Bootstrap 
curl -fsSL https://raw.githubusercontent.com/Senior3514/MultiClaw/main/scripts/bootstrap.sh | bash
2. Configure 
multiclaw configure
3. Start 
multiclaw up --provider openai --model gpt-5.4 --api-key YOUR_KEY
4. Verify 
multiclaw status
Helpful commands
multiclaw walkthrough
multiclaw guide
multiclaw stop
If you need to remove it
curl -fsSL https://raw.githubusercontent.com/Senior3514/MultiClaw/main/scripts/uninstall.sh | bash
